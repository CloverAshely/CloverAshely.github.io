java基础篇之JDBC

发表于 2018-05-11

JDBC：1.官网下载 connect J 压缩包解压
2.打开idea（CTRL+ AIT+SHIFT+s）的project structure，Modules ————> dependencies 添加JARS or directories,选择jar包路径
3.编辑 private static String driver=”com.mysql.cj.jdbc.Driver”;//获得驱动类
private static String url=”jdbc:mysql://localhost:3306/yiibaidb?serverTimezone=UTC&useSSL=false”;//数据库地址资源。后面新增版本变更，不加会警告
private static String user=”root”;
private static String password=”123456”;
4.创建连接时，需要3个接口：connection ,PreparedStatement,ResultSet
PreparedStatement:预处理
Statement：批处理时使用
CallableStatement：用于调用存储过程
connection：建立数据库的连接
ResultSet：结果集

关于数据库的基本操作语句：
. 查询 : 查询所有数据 : select * from table_name;
.添加数据 : Insert into 表名 [(字段1 , 字段2 , ….)] values (值1 , 值2 , …..);
Update table_name set 字段名=’新值’ [, 字段2 =’新值’ , …..][where id=id_num] [order by 字段 顺序]
　 删除整个表中的信息 : delete from table_name;
　　 删除表中指定条件的语句 : delete from table_name where 条件语句 ; 条件语句如 : id=3;
