pagejava基础学习之MySQL安装

Posted on 2018-05-10

1.从官网下载压缩包解压的方式来安装MySQL，因为是比较新版5.7.21，bin同级目录下没有data文件夹，而且找不到my.ini和default.ini配置文件
2.解压之后进入安装根目录下自己编写my.ini配置文件。配置环境变量path
3.以管理员身份进入cmd。cd到bin下，命令：mysqld -install———>mysqld –initialize –user=mysql –console(初始化创建data文件夹)————>net start mysql(启动数据库)
————> set password for root@localhost=password(‘你的密码’);{设置密码}————> show database————>\g;(关闭数据库)
————>net stop mysql
数据库的命令行后面加 ; 结束

        关于数据库操作的工具 Navicat 破解补丁：
        1. 打开破解补丁目录：，打开命令提示符或者命令提示符(管理员) ，建议使用命令提示符(管理员) 最为稳妥
        2.用Patch.exe给Navicat的主程序即navicat.exe打个补丁，换掉其中的公钥，
        执行Patch.exe <navicat.exe path>命令，<navicat.exe path>就是navicat.exe的完整路径，如Patch.exe "C:\Software\Navicat Premium 12\navicat.exe"
        3.接着用Keygen.exe生成Navicat 12的注册码，执行Keygen.exe RegPrivateKey.pem命令，根据提示输入用户名和组织名，将生成了注册码，
        4.断开网络连接（直至激活成功），打开Navicat，手动激活
        5.将请求码复制到命令提示符中并回车（若未生成激活码请再次回车），将生成激活码
        6.将生成的激活码复制到 Navicat激活码输入框中，点击激活
        补充：若要升级版本，则需要删除Navicat安装目录的navicat.exe.backup和补丁文件夹下的RegPrivateKey.pem
